# Менеджер файлов

Позволяет копировать файлы, удалять каталоги/файлы,
подсчитывать количество файлов и размер каталога и его содержимого, осуществлять поиск файлов в каталоге.

Для начала работы необходимо ввести в текстовое поле слева абсолютный путь к файлу/каталогу или воспользоваться кнопками выбора.

После этого можно использовать следующие функции:
- `Количество файлов` подсчитывает общее количество файлов в каталоге и во всех вложенных подкаталогах;
- `Размер файлов` покажет размер выбранного каталога и размер каждого из отдельных файлов в нём, размер вложенных каталогов на один уровень вниз;
- `Копировать файл` копирует выбранный файл в тот же каталог, добавляет к имени файла `copy` (если копия уже существует, создаст ещё одну с добавкой `another_copy`);
- `Поиск файлов` выведет пары каталог-файл из выбранного каталога согласно подстроке или регулярному выражению из поля рядом (если поле оставить незаполненным, выведет все пары файлов);
- `Добавить дату` добавит дату создания файла к имени файла (если выбран конкретный файл) или к именам всех файлов в выбранном каталоге (если включена опция `Включая вложенные`, то изменения будут применены ко всем файлам во вложенных каталогах);
- `Удалить файлы` удаляет файл/каталог минуя корзину. Использовать с осторожностью. Перед употреблением включить.
